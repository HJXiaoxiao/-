# 操作系统大作业

### 系统环境和库函数

- Windows
- Dev-C++
- Easyx库

### 课程内容设计

编写程序完成内存可变分区分配模拟仿真，程序运行后可使用五种算法：首次适应，下次适应，最佳适应，最坏适应，快速分配。要求输入仿真界面清晰直观，开发语言不限制。

设置一个数组，长度为1000B，当作内存。通过随机设置四个作业容量，按照以上五个算法进行空间分配，显示分配结果。例如如J1为50B，J2为80B，J3为240B，J4为300B。
